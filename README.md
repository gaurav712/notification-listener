# Notification Reader

An Android app that reads and displays Google Maps navigation notifications.

## Features

- Reads navigation notifications from Google Maps
- Displays title, subtitle, progress, and icon
- Real-time updates during navigation

## Setup

1. Build and install the app
2. Grant notification listener permission when prompted
3. Start navigation in Google Maps to see updates

## Requirements

- Android 7.0 (API 24) or higher
- Notifications permission
